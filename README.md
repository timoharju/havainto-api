# havainto-api

```https://europe-west1-havainto-api.cloudfunctions.net/api/havainnot```

```https://europe-west1-havainto-api.cloudfunctions.net/api/havainnot/::havaintoID```

```https://europe-west1-havainto-api.cloudfunctions.net/api/havainnot/```

API DELETE yhdelle havainnoille

```https://europe-west1-havainto-api.cloudfunctions.net/api/havainnot/::havaintoID```
